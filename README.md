# ReLife

Çok kiracılı (multi-tenant) hayvan barınağı bağış platformu. Barınaklar kendi hayvanlarını
ve ihtiyaçlarını yönetir; bağışçılar tek hesapla platform genelinde istedikleri hayvana
bağış yapar, rozet kazanır ve global sıralamada yer alır.

## Teknoloji

TALL stack — **Tailwind + Alpine.js + Laravel 11 + Livewire 3**, yönetim panelleri için
**Filament 3**.
